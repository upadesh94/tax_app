# Estate Tax Collection System

A modern React Native mobile app for managing property taxes and payments.

## Quick Start

```bash
npm install
npx expo start
```

## Features

- Property management
- Tax calculation (Property & Income)
- Payment processing
- Dark/Light mode
- Real-time data from Supabase
